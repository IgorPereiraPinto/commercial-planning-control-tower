# Build DRE

## Objetivo
Atalho para montar DRE gerencial a partir de lançamentos, categorias ou dados brutos.

## Quando usar
Use este command quando o usuário tiver lançamentos financeiros ou categorias de receita
e despesa e precisar transformar isso em uma DRE estruturada com indicadores.

## Instrução de execução
Conduza a resposta nesta ordem:
1. entender a fonte dos dados (lançamentos, razão contábil, planilha)
2. classificar cada linha na estrutura DRE (Receita Bruta, Deduções, CPV, etc.)
3. montar a DRE com subtotais: Receita Líquida, Lucro Bruto, EBITDA, Lucro Líquido
4. calcular margens: bruta, EBITDA, EBIT, líquida
5. comparar com período anterior ou orçado (se disponível)
6. identificar as 3 linhas com maior impacto positivo e negativo
7. traduzir em narrativa executiva de 2-3 parágrafos

## Formato de saída
1. DRE estruturada (tabela com R$ e margens %)
2. KPIs principais (EBITDA, margens)
3. variações vs período anterior ou budget
4. destaques positivos e negativos
5. narrativa executiva (contexto → achado → implicação)
6. próximos passos
