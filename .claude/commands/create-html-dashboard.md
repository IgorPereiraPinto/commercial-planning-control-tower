# Create HTML Dashboard

## Objetivo
Atalho para criar dashboards web analíticos em HTML, CSS e JavaScript.

## Quando usar
Use este command quando o usuário pedir painel web, dashboard HTML para portfólio, protótipo executivo ou adaptação de painel para navegador.

## Instrução de execução
Conduza a resposta nesta ordem:
1. definir objetivo do painel
2. estruturar filtros, KPIs e visuais
3. organizar a arquitetura do front-end
4. gerar o código HTML/CSS/JS
5. indicar como adaptar para dados reais

## Formato de saída
1. resumo da arquitetura
2. estrutura do dashboard
3. código completo
4. lógica de filtros
5. observações para manutenção
6. como adaptar para dados reais
