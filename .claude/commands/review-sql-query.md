# Review SQL Query

## Objetivo
Atalho para revisar, explicar, corrigir ou otimizar queries SQL.

## Quando usar
Use este command quando o usuário colar uma query, relatar erro, pedir otimização ou quiser transformar uma necessidade em SQL.

## Instrução de execução
Conduza a resposta nesta ordem:
1. entender o objetivo da query
2. apontar problemas de lógica, performance ou legibilidade
3. propor versão ajustada
4. explicar principais mudanças
5. sugerir validações

## Formato de saída
1. entendimento da necessidade
2. diagnóstico da query
3. SQL revisado
4. explicação das melhorias
5. validações sugeridas
6. observações de performance
