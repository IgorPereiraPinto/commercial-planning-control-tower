# Improve Professional Text

## Objetivo
Atalho para humanizar, revisar e melhorar textos profissionais.

## Quando usar
Use este command quando o usuário pedir melhoria de e-mail, relatório, mensagem, texto executivo ou documentação.

## Instrução de execução
Conduza a resposta nesta ordem:
1. preservar a intenção original
2. melhorar clareza e fluidez
3. ajustar o tom ao canal
4. remover jargões e construções artificiais
5. entregar versão revisada
6. oferecer alternativa mais curta ou mais formal, quando útil

## Formato de saída
1. versão revisada
2. ajustes de tom
3. versão alternativa
4. observações de clareza e impacto
