# Design Dashboard

## Objetivo
Atalho para estruturar dashboards, definir KPIs, layout, filtros e visuais.

## Quando usar
Use este command quando o usuário pedir ajuda para montar um dashboard conceitualmente, sem necessariamente gerar código.

## Instrução de execução
Conduza a resposta nesta ordem:
1. entender objetivo do painel
2. identificar público e decisão
3. definir KPIs principais
4. sugerir layout e seções
5. recomendar visuais
6. indicar filtros
7. destacar narrativa do painel

## Formato de saída
1. objetivo do painel
2. público e decisão esperada
3. KPIs principais
4. estrutura visual sugerida
5. visuais recomendados
6. filtros
7. observações de UX e narrativa
