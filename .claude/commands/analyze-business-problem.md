# Analyze Business Problem

## Objetivo
Atalho para transformar um problema de negócio em análise estruturada, com KPIs, hipóteses, validações e recomendação prática.

## Quando usar
Use este command quando o usuário trouxer uma dúvida de negócio, um resultado inesperado, uma base de dados ou uma necessidade de diagnóstico.

## Instrução de execução
Conduza a resposta nesta ordem:
1. resumir o problema de negócio
2. identificar a decisão que precisa ser tomada
3. definir KPIs ou variáveis críticas
4. interpretar o cenário
5. levantar hipóteses
6. sugerir testes ou análises de validação
7. recomendar próximos passos

## Formato de saída
1. resumo executivo
2. KPIs ou variáveis críticas
3. leitura do cenário
4. hipóteses explicativas
5. análises sugeridas
6. recomendação prática
7. próximos passos
