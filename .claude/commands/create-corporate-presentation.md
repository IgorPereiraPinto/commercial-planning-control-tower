# Create Corporate Presentation

## Objetivo
Atalho para estruturar apresentações corporativas com mensagem por slide e sugestão visual.

## Quando usar
Use este command quando o usuário pedir uma apresentação executiva, gerencial ou consultiva.

## Instrução de execução
Conduza a resposta nesta ordem:
1. identificar objetivo e público
2. definir mensagem central
3. montar outline dos slides
4. escrever a mensagem principal por slide
5. sugerir visual adequado por slide
6. indicar fala de apoio, quando útil

## Formato de saída
1. objetivo da apresentação
2. público e decisão esperada
3. mensagem central
4. outline dos slides
5. mensagem principal por slide
6. sugestão visual por slide
7. observações para apresentação oral
