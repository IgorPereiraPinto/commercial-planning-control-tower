# Build Executive Summary

## Objetivo
Atalho para gerar sumários executivos curtos, claros e orientados à decisão.

## Quando usar
Use este command quando o usuário pedir um resumo para diretoria, one-pager, abertura de relatório ou síntese gerencial.

## Instrução de execução
Conduza a resposta nesta ordem:
1. contextualizar o tema em poucas linhas
2. destacar os principais achados
3. traduzir impacto no negócio
4. indicar decisão sugerida
5. fechar com próximos passos

## Formato de saída
1. contexto
2. achados principais
3. impacto no negócio
4. decisão sugerida
5. próximos passos
