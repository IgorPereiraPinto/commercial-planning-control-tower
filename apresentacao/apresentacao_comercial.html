Acurácia do Forecast
MAPE como KPI de processo: medimos o erro para melhorar o planejamento
O que é o MAPE?
Mean Absolute Percentage Error
Erro percentual médio absoluto entre o forecast e o realizado.
<10%
Excelente
10–20%
Boa
20–50%
Razoável
>50%
Baixa
Como calculamos?
MAPE =
AVERAGEX(
  Meses × Vendedores,
  ABS(Real − Meta)
  / ABS(Real)
)
Real = Faturamento · Meta = Forecast
Por que isso importa?
📌MAPE > 25% = o processo de planejamento precisa de revisão
📌Não é problema do vendedor — é do modelo de metas
📌Medir é o primeiro passo para melhorar
🎯Meta do projeto: MAPE < 15%
